# How to Start the Frontend

## Quick Start

The frontend is in the `trading-dashboard` directory. You need to navigate there first:

### Option 1: Navigate and Run
```powershell
cd "trading-dashboard"
npm run dev
```

### Option 2: One-Line Command
```powershell
cd trading-dashboard; npm run dev
```

### Option 3: From Root Directory
```powershell
cd "D:\blackhole projects\blackhole-infevers trade\Multi-Asset Trading Dashboard\trading-dashboard"
npm run dev
```

## Full Path
The package.json is located at:
```
D:\blackhole projects\blackhole-infevers trade\Multi-Asset Trading Dashboard\trading-dashboard\package.json
```

## If npm is not installed
If you get an error about npm, make sure Node.js is installed:
1. Download from: https://nodejs.org/
2. Install it
3. Restart your terminal
4. Run `npm --version` to verify

## If dependencies are not installed
If you get module errors, install dependencies first:
```powershell
cd trading-dashboard
npm install
npm run dev
```
